# Feedback

## Installation

Feedback is available through Jitpack.io

```groovy
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    compile 'com.github.loup-studio:LPM-Feedback-android:master-SNAPSHOT'
}
```
